# Recency/Latency 리텐션 강의 HTML 슬라이드

## 파일

- `index.html` — 브라우저에서 바로 열 수 있는 HTML 슬라이드

## GitHub에서 확인하는 방법

### 방법 A. GitHub Pages 사용

1. GitHub에 새 repository를 만든다.
2. 이 폴더 안의 `index.html`과 `README.md`를 repository 루트에 업로드한다.
3. Repository Settings → Pages로 이동한다.
4. Source를 `Deploy from a branch`로 선택한다.
5. Branch를 `main` / folder를 `/root`로 선택한다.
6. 저장 후 GitHub Pages URL에서 확인한다.

### 방법 B. GitHub 웹 UI에서 파일만 확인

1. `index.html`을 GitHub repository에 업로드한다.
2. 파일을 클릭한다.
3. Raw 또는 Download로 내려받아 브라우저에서 연다.

## 조작법

- 오른쪽 화살표 / Space: 다음 슬라이드
- 왼쪽 화살표: 이전 슬라이드
- Home: 첫 슬라이드
- End: 마지막 슬라이드
- Speaker Notes는 각 슬라이드 하단의 접이식 영역에서 확인 가능
